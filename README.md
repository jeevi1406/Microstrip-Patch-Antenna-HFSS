# 📡 Dual-Band Slotted Microstrip Antenna for Smart Farming Communication Sensor

## 🌾 Project Overview
A compact dual-band slotted microstrip patch antenna designed for IoT-based 
smart agricultural communication systems. Designed and simulated using ANSYS HFSS.

## 🎯 Operating Frequencies
| Band | Frequency | Return Loss | VSWR |
|------|-----------|-------------|------|
| Band 1 | 2.4 GHz | -29.43 dB | 1.10 |
| Band 2 | 4.29 GHz | -35.77 dB | 1.13 |

## 🔧 Antenna Specifications
- **Antenna Type:** Slotted Rectangular Microstrip Patch Antenna
- **Substrate Material:** FR4 (εr = 4.4)
- **Substrate Thickness:** 1.6 mm
- **Patch Material:** Copper
- **Feeding Technique:** Microstrip Line Feed
- **Patch Dimensions:** W = 38 mm, L = 29.44 mm
- **Simulation Tool:** ANSYS HFSS 2025 R2

## ✅ Key Results
- Return Loss of **-29.43 dB** at 2.4 GHz and **-35.77 dB** at 4.29 GHz
- VSWR close to unity at both frequencies (1.10 and 1.13)
- Impedance ≈ 50Ω at both bands (48.07Ω and 49.75Ω)
- Quasi-omnidirectional radiation pattern for wide field coverage
- Dual-band achieved through strategic slot design (no size increase)

## 🌱 Application
Smart agriculture IoT systems including:
- Soil moisture monitoring
- Crop health surveillance
- Automated irrigation control
- Wireless sensor node communication
